# Ful Stack Developer Test

To get JSON from https://jsonplaceholder.typicode.com/comments by sending a request and then write the data to Database.

## Running the Project

Use the Dockerfile.
Incase of failure to build, clone the project and cd to directory.
then use,
```bash
pip install -r requirements.txt
```

## Usage

```bash
python ./main.py
```
